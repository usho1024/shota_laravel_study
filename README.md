# shota_laravel_study

## 概要
Laravelの基本をハンズオンで学ぶための掲示板アプリです。

## 目的
- Laravelで基本的なCRUD操作+αができるようになる。
- 具体的には公式リファレンスの「はじめに〜テスト」の内容を自分で説明できるレベルまで到達する。

## 初期設定
 
 ```bash or zsh
 $ cp .env.example .env
 
 $ composer install
 $ docker compose build
 $ docker compose up -d
 $ php artisan migrate
 ```
